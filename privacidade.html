Política de Privacidade do Tagged

No Tagged, levamos sua privacidade a sério. Esta política explica como coletamos, usamos, armazenamos e protegemos seus dados pessoais.

💡 Compromisso: Nunca vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins comerciais.

1. Dados que Coletamos

Dados de Cadastro Básico:

  • Email (para autenticação)
  • Nickname/Apelido (identificação pública)
  • Senha (criptografada)

Dados de Cadastro Completo (opcional inicialmente):

  • Nome completo
  • CPF (obrigatório para assinar petições)
  • Telefone/WhatsApp
  • Foto de perfil (opcional)
  • Biografia (opcional)
  • Localização (cidade/estado e, com seu consentimento, GPS para marcar denúncias no mapa)

Permissões do dispositivo (com seu consentimento):

  • Câmera e fotos — para anexar evidências, foto de perfil e capas
  • Microfone — para gravar áudios em mensagens e conteúdos
  • Localização — para registrar onde ocorre uma denúncia/evento (nunca em segundo plano)

Dados de Uso:

  • Denúncias criadas
  • Petições assinadas
  • Comentários publicados
  • Interações (likes, compartilhamentos)
  • Mensagens do chat

2. Como Usamos Seus Dados

Usamos seus dados exclusivamente para:

  • Autenticar seu acesso à plataforma
  • Exibir seu perfil público (nickname, foto, biografia)
  • Validar assinaturas de petições (nome completo e CPF)
  • Facilitar comunicação entre usuários (chat)
  • Melhorar a experiência e funcionalidades da plataforma
  • Enviar notificações importantes sobre suas denúncias e agendamentos
  • Prevenir fraudes e abuso da plataforma

3. Compartilhamento de Dados

Seus dados pessoais são compartilhados apenas em situações específicas:

Dados Públicos (visíveis para todos):

  • Nickname
  • Foto de perfil
  • Biografia
  • Localização (cidade/estado)
  • Denúncias não-anônimas
  • Comentários públicos

Dados de Petições (compartilhados com autoridades):

  • Nome completo dos signatários
  • CPF dos signatários
  • Data da assinatura

Nota: Estes dados são necessários para dar validade legal às petições.

Dados Privados (nunca compartilhados):

  • Email
  • Senha
  • Telefone (exceto se você escolher tornar público)
  • Mensagens privadas do chat
  • Denúncias anônimas (autor não é revelado)

Provedores de infraestrutura:

  • Google Firebase — armazenamento, autenticação e banco de dados
  • Expo — entrega de notificações push

4. Armazenamento e Segurança

  • Usamos Firebase (Google) para armazenamento
  • Senhas são criptografadas (hash seguro)
  • Conexões protegidas por HTTPS/SSL
  • Backups regulares para prevenir perda de dados

5. Seus Direitos (LGPD)

De acordo com a LGPD (Lei nº 13.709/2018), você tem direito a:

  • Acessar todos os seus dados pessoais
  • Corrigir dados incorretos ou desatualizados
  • Solicitar exclusão de sua conta e dados
  • Revogar consentimentos dados anteriormente
  • Exportar seus dados em formato legível
  • Opor-se ao processamento de dados em certas circunstâncias

Para exercer esses direitos, use o menu Configurações → Editar Perfil dentro do app, ou envie um email para privacidade@apptagged.com. Veja também a página de Exclusão de Dados.

6. Cookies e Tecnologias Similares

  • Cookies de sessão (manter você logado)
  • Armazenamento local (preferências do app)
  • Analytics (entender como o app é usado)

7. Menores de Idade

O Tagged é destinado a maiores de 16 anos. Se você tem entre 16 e 18 anos, recomendamos que use a plataforma com supervisão de um responsável legal.

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Notificaremos mudanças significativas através da plataforma. Recomendamos revisar esta página regularmente.

9. Contato — Encarregado de Dados (DPO)

  • Email: privacidade@apptagged.com
  • Dentro do app: Menu → Configurações → Contato
🔒 Sua privacidade é nossa prioridade. Se tiver dúvidas sobre como seus dados são usados, não hesite em nos contatar.
